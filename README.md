# OCR-QuestGen-FreeAI

OCR-QuestGen-FreeAI is project made with passion in Bangladesh as a way to optimize AI question paper generation for free and user-friendly so AI-powered question paper and mark-scheme generation so that all students are able to use technology to achieve their full academic potential.

OCR-QuestGen-FreeAI হলো বাংলাদেশে গভীর আগ্রহ থেকে তৈরি একটি প্রকল্প, যার লক্ষ্য হলো বিনামূল্যে ও ব্যবহারবান্ধব উপায়ে কৃত্রিম বুদ্ধিমত্তার (AI) মাধ্যমে প্রশ্নপত্র ও নম্বর বিভাজন তৈরি করা, যাতে সকল শিক্ষার্থী প্রযুক্তি ব্যবহার করে তাদের শিক্ষাগত সম্ভাবনার পূর্ণ বিকাশ ঘটাতে পারে।
